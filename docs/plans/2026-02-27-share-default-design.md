# 首页分享图设计文档

**日期**: 2026-02-27
**功能**: 首页推荐分享图 (share-default.png)
**设计方向**: 现代简洁风格，保持设计语言一致

---

## 设计目标

生成一张用于推荐晨读营给朋友的分享图，采用**现代简洁风格**，与既有的 share-insight.png 保持设计语言一致。

## 规格

| 项目 | 规格             |
| ---- | ---------------- |
| 尺寸 | 1080×1080 px     |
| 用途 | 首页推荐分享卡片 |
| 风格 | 现代简洁         |

## 配色方案

- **背景渐变**：与 share-insight.png 完全一致
  - 顶部: #5B9FE3（亮蓝）
  - 中部: #4A8FD8（蓝）
  - 底部: #3D7BC7（深蓝）
- **主文字**：#FFFFFF（白色）+ 轻微黑色投影
- **副文字**：#FFFFFF（白色），略微降低不透明度

## 排版设计

### 主标题：「凡人晨读营」

- **排列方式**：竖排（5个字上下排列）
- **字体大小**：160px
- **字体**：STHeiti Medium
- **字间距**：40-60px
- **位置**：垂直居中，水平居中
- **效果**：黑色投影 rgba(0,0,0,0.15)

### 副文案：「每天晨读 · 遇见更好的自己」

- **排列方式**：横排
- **字体大小**：36px
- **字体**：STHeiti Light
- **颜色**：白色，略微降低不透明度
- **位置**：主标题下方 80-100px
- **效果**：轻盈、优雅

## 装饰元素

**分隔线**（可选）：

- 位置：主标题和副文案之间
- 样式：细的白色横线，长度 400-500px
- 透明度：中等
- 用途：轻微分隔，不显眼

**其他装饰**：无（极简原则）

## 实现细节

### Python 脚本修改

修改 `generate_share_image.py` 中的 `generate_default_image()` 函数：

- 使用相同的蓝色渐变背景
- 竖排排列"凡人晨读营"五个字
- 横排显示副文案
- 去除之前的太阳光晕装饰
- 保持极简风格

### 验证方式

1. 运行脚本生成 share-default.png
2. 在 share-image-generator.html 中预览效果
3. 确认排版清晰、优雅
4. 提交到 GitHub
