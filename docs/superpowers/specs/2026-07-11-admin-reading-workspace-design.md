# 管理后台晨读会工作台设计

## 设计结论
采用“书房工作台”方向。登录页继续以摊开的书承载品牌氛围；登录后的后台用书脊式侧栏、章节眉题、暖白纸张、细分隔线和克制的品牌色强调承接同一气质，同时保持高频运营页面的密度与扫描效率。

## 视觉层级
1. 侧栏像一本书的封面与目录：租户名称、晨读营标识、分组目录和当前章节清晰可见。
2. 顶栏像书页眉题：显示所属业务章节、当前页面标题和简短说明，租户切换与账号入口保持在右侧。
3. 内容区像工作笔记纸：使用中性暖白底、轻微纸张纹理和稳定的内容宽度，不把页面包成装饰卡片。
4. 业务组件像索引卡：卡片、筛选、表格、标签和弹窗使用统一的 8px 以内圆角、细边框与轻阴影。

## 租户规则
租户配置的 `primaryColor` 只用于按钮、选中态、焦点、数据重点和浅色强调面。纸张、墨色、边框和语义状态色保持共享，避免不同租户变成完全不同的后台产品。无效色值回退到默认晨读绿。

## 行为边界
现有路由、权限、业务接口、表格字段、筛选行为、数据库二次验证和租户切换行为全部保持不变。本次不新增后端接口或数据迁移。

## 验收
- 登录页与登录后后台在字体、色调、层级和细节上属于同一设计语言。
- 代表性页面（仪表板、数据分析、报名管理、内容管理、租户管理）无明显风格断裂。
- 默认绿色与至少一个非绿色租户色均保持可读。
- 1440px 桌面和 860px 以下窄屏无遮挡、不可达控件或非预期横向溢出。
