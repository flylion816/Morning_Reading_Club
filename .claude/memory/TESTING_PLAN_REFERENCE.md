# 历史测试计划参考

本文件保留为轻量入口，用于指向历史测试建设计划，不再重复完整计划正文。

## 当前用途

- 当需要恢复测试建设上下文时，先看这里
- 当只需要执行现有测试时，不要优先打开本文件

## 主要入口

- 测试计划索引：`docs/plans/testing/TESTING_INDEX.md`
- 测试分析：`docs/plans/testing/TESTING_ANALYSIS.md`
- 测试快速开始：`docs/plans/testing/TESTING_QUICK_START.md`
- 测试总指南：`docs/guides/TESTING_GUIDE.md`

## 适合查这个文件的场景

- 想知道 Admin / Miniprogram 测试建设曾经如何拆阶段
- 想追溯历史测试目标和完成度口径
- 想把旧计划迁移到新的测试任务里

## 不适合查这个文件的场景

- 只想运行测试
- 只想补一个回归测试
- 只想确认某个命令怎么跑

这些场景直接去 `docs/guides/TESTING_GUIDE.md` 或具体模块测试目录。
