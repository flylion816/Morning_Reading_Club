晨读营每日业务全景，按时间线组织：

1. 前置：学员浏览期次 → 报名（付费/免费）→ 收到报名结果通知
2. 05:45 唤醒：系统推送微信订阅消息 → 学员进入小程序 → 查看今日课节（冥想词/问题/音频/行动）→ 晨读（06:00–06:10）
3. 晨读后打卡：提交打卡（时长/完成率/心情/笔记/图片）→ 积分累计 → 排行榜更新
4. 小凡看见下发：运营者预先撰写点评 → 系统晨读结束后推送给学员 → 学员查看/申请他人看见/发弹幕
5. 社区互动：打卡广场浏览、评论、点赞
6. 运营管理：后台撰写看见、评论打卡、删除不合适内容、审批看见申请
7. 所有互动均通过站内通知 + 微信订阅消息触达学员
