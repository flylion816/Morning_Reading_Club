## ADDED Requirements

### Requirement: 管理后台上传结营视频
系统 MUST 允许管理员在课节编辑中为结营词课节上传一个结营视频，并保存视频地址、原始文件名、文件大小、时长、首帧封面图和上传时间。

#### Scenario: 管理员上传结营视频成功
- **GIVEN** 管理员已登录管理后台并选择了具体租户
- **AND** 管理员正在编辑最后一课结营词课节
- **WHEN** 管理员选择符合格式和大小限制的视频文件并保存课节
- **THEN** 系统保存该课节的结营视频信息
- **AND** 课节详情接口返回该结营视频信息。

#### Scenario: 管理员删除结营视频
- **GIVEN** 某结营词课节已配置结营视频
- **WHEN** 管理员在课节编辑中删除该视频并保存
- **THEN** 系统清空该课节的结营视频信息
- **AND** 小程序课程详情不得再展示结营视频板块。

#### Scenario: 非视频文件被拒绝
- **GIVEN** 管理员在结营视频上传控件中选择文件
- **WHEN** 文件不是系统允许的视频格式
- **THEN** 系统拒绝上传
- **AND** 系统提示仅支持视频文件。

### Requirement: 结营视频首帧封面
系统 MUST 在上传结营视频时生成或保存一张可用于小程序展示和分享卡片的首帧封面图；当首帧封面不可用时，系统 MUST 回退到默认课程分享封面。

#### Scenario: 首帧封面生成成功
- **GIVEN** 管理员上传了可被浏览器读取首帧的视频
- **WHEN** 上传流程完成
- **THEN** 系统保存视频首帧封面图地址
- **AND** 小程序视频播放器使用该封面作为 poster。

#### Scenario: 首帧封面生成失败
- **GIVEN** 管理员上传的视频无法生成首帧封面
- **WHEN** 管理员保存课节
- **THEN** 系统仍可保存视频信息
- **AND** 小程序分享结营词时使用默认课程分享封面。

### Requirement: 小程序播放结营视频
系统 MUST 在小程序课程详情页检测到课节存在结营视频时展示“结营视频”板块，并允许用户直接播放视频。

#### Scenario: 有结营视频时展示播放器
- **GIVEN** 用户打开包含结营视频的结营词课节详情页
- **WHEN** 课节详情加载完成
- **THEN** 页面展示“结营视频”板块
- **AND** 页面展示可播放的视频播放器
- **AND** 播放器封面优先使用视频首帧封面。

#### Scenario: 无结营视频时不展示播放器
- **GIVEN** 用户打开未配置结营视频的课节详情页
- **WHEN** 课节详情加载完成
- **THEN** 页面不得展示“结营视频”板块。

### Requirement: 分享结营视频和结营词
系统 MUST 支持用户从结营视频板块分享微信小程序页面，并在分享结营词时优先使用结营视频首帧封面作为分享卡片封面。

#### Scenario: 分享结营视频
- **GIVEN** 用户正在查看包含结营视频的结营词课节
- **WHEN** 用户点击结营视频板块内的分享按钮
- **THEN** 微信分享卡片标题体现结营视频和课程标题
- **AND** 分享路径指向该课程详情页并携带定位到结营视频板块的参数
- **AND** 分享卡片封面使用视频首帧封面。

#### Scenario: 分享结营词使用视频首帧
- **GIVEN** 用户正在查看包含结营视频首帧封面的结营词课节
- **WHEN** 用户触发课程详情页普通分享
- **THEN** 分享卡片标题使用课程标题
- **AND** 分享卡片封面使用结营视频首帧封面。

#### Scenario: 从分享进入定位视频板块
- **GIVEN** 用户通过结营视频分享卡片进入小程序
- **WHEN** 课程详情页加载完成
- **THEN** 页面自动定位到结营视频板块。
