## ADDED Requirements

### Requirement: Immersive Reading Mode
The system SHALL provide a full-screen reading mode for the course "读一读" module.

#### Scenario: Enter reading mode
- **WHEN** a learner taps the immersive reading entry on course detail
- **THEN** the system opens a focused reading screen containing the current section reading content.

### Requirement: Current Moment Reading Poster
The system SHALL allow the learner to generate a shareable poster for the current reading position.

#### Scenario: Share current reading moment
- **WHEN** the learner taps "分享此刻"
- **THEN** the system generates a poster containing the period name, lesson title, surrounding reading text, an enlarged current excerpt, "凡人共读", "一个早起、读书、谈心的地方", and the mini-program code.
